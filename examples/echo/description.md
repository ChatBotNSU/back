# echo

Минимальный пример. Главный граф крутится в цикле `text_answer → set_message → send_message`, без сабграфов, без условий. Бот повторяет последнее сообщение пользователя.

**Что демонстрирует:**

- Базовый цикл: одна итерация графа = одно сообщение пользователя → одно сообщение бота.
- `text_answer` создаёт переменную динамически — никакого `Chatbot.variables` объявлять не нужно.
- После `send_message` граф возвращается на `n_read` и ждёт следующего входящего сообщения.

**Ожидаемое поведение:**

```
user > привет
bot  < Ты написал: привет

user > как дела
bot  < Ты написал: как дела
```
